# Numerical Methods & Optimization

This repository contains professional MATLAB implementations of core numerical algorithms developed during my engineering coursework. The project focuses on creating reusable, efficient, and well-documented functions for scientific computing.

---

## 📁 Repository Structure

### 🔹 Numerical Methods 1
Focuses on fundamental algorithms for solving linear and non-linear systems, as well as data interpolation.
* **Root Finding & Linear Systems**: Iterative solvers and convergence analysis (e.g., Jacobi, Gauss-Seidel).
* **Interpolation**: Polynomial approximation techniques (e.g., Newton and Lagrange polynomials).

### 🔹 Numerical Methods 2
Advanced algorithms tackling numerical calculus, integration, and differential equations.
* **Numerical Differentiation & Integration**:
  * Richardson extrapolation for high-accuracy derivatives.
  * Romberg integration and composite rules (Simpson, Trapezoidal).
  * Gaussian quadrature utilizing orthogonal polynomials and the **Jacobi matrix** (eigenvalue/eigenvector evaluation).
* **Ordinary Differential Equations (ODEs)**:
  * Euler's method for coupled systems of differential equations.
  * High-order Runge-Kutta methods configured via the **Butcher tableau**.
